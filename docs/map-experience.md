# Map Experience Spec

> Silmaril의 **핵심 경험 = Map**. 내부 개념 Atlas 를 사용자에게 보여주는 화면.
> 정본 방향: [silmaril-v2-direction.md](./silmaril-v2-direction.md) · 지식 모델: [canonical-knowledge-model.md](./canonical-knowledge-model.md).
> 원칙: 저장-first · 기록은 선택 · 연결 그래프가 핵심 자산 · 사용자 용어는 **미발견/미확인 실마리/새로운 흔적**(`Fog`/`Locked`/`???` 금지).

## 0. 한 줄

Map은 "내가 밝힌 세계"를 보여주고, 그 가장자리의 **새로운 흔적**으로 계속 탐험하게 만드는 화면이다.
사용자는 정보를 정리하지 않는다 — **세계를 밝혀간다.**

## 1. 3개 레이어

```
┌──────────────────────────────────────────┐
│              ☀  하늘 (Sky)                 │  ← 나침반: 방향/추천/발견
│         [ Silmaril 아이콘 ]                │
├──────────────────────────────────────────┤
│                                            │
│              🌊 바다 (Sea)                  │  ← Active Map: 노드·연결·탐험
│        ●───●   ◌(미발견)                   │
│        │   ╲                               │
│        ●    ●───◌                          │
│                                            │
├──────────────────────────────────────────┤
│           🏝  땅 (Land)                     │  ← Bottom Sheet: 선택 상세·행동
│   [ 디터 람스 ]  저장 · 컬렉션 · 연결 …     │
└──────────────────────────────────────────┘
```

### 1.1 하늘 (Sky) — 나침반
- **상단 중앙 Silmaril 아이콘.** 길을 잃지 않게 하는 기준점(나침반).
- 역할:
  - **추천 실마리** 제시 (최근 저장/연결 기반).
  - **미발견 영역으로 이동** (그래프 가장자리의 새로운 흔적으로 카메라 이동).
  - **Daily Discovery** (오늘의 실마리 한 개 — 선택 마비 제거).
- 인터랙션: 아이콘 탭 → 추천/오늘의 발견/미발견 점프의 가벼운 메뉴(또는 시트). 평소엔 조용히 떠 있음.

### 1.2 바다 (Sea) — Active Map
- 화면의 본체. **Atlas 데이터**를 시각화.
- 요소:
  - **Thread 노드** — 밝혀진(저장/조회) 실마리. 타입별 미세한 구분.
  - **Connection** — 노드 간 엣지. Tier 1(사실)=옅은 실선, Tier 2(해석)=accent 색 (D-016/연결모델).
  - **미발견 실마리** — 저장 노드의 이웃 중 아직 안 본 것. 흐릿한 미점등 노드("새로운 흔적"). 절대 `???` 표기 안 함.
  - **탐험 경로** — 사용자가 따라온 흐름(최근 이동) 살짝 강조.
- 인터랙션:
  - **키워드/노드 탭** → 카메라가 그 노드로, 이웃 펼침. **스와이프**로 인접 실마리 훑기 (D-015).
  - 미발견 노드 탭 → 드러나며(미발견→발견) 그 이웃이 새 흔적으로 확장 → 호기심 루프.
  - MVP는 가벼운 노드/카드 구조(무거운 그래프 엔진 X, 3D Globe X).

### 1.3 땅 (Land) — Bottom Sheet
- 노드 선택 시 **아래에서 올라오는 시트**(페이지 이동 아님 → 탐험 끊김 최소화).
- 내용(위→아래):
  - 실마리 **요약**(summary) + 타입.
  - **저장** 버튼 (1초, save-first, 1급 행동).
  - **컬렉션 추가**.
  - **관련 실마리** (연결 리스트 — Tier 표시, 탭하면 Sea의 카메라가 그 노드로).
  - **관점**(Perspective) — Should-have, 가볍게(없으면 자리만).
  - **출처**(sources).
- 시트는 3단(peek/half/full) 스냅. peek=요약+저장, half=연결, full=관점·출처.

## 2. Map UX Flow

```
앱 진입 → Map (Sea 중앙 = 내가 밝힌 세계)
  │
  ├─ Sky 아이콘 탭 → 오늘의 발견 / 추천 / 미발견 점프
  │
  ├─ 노드 탭 → Land 시트(peek: 요약+저장)
  │     ├─ 저장 → 노드 점등, Sky에 "한 칸 더 밝아짐" 피드백, Archive에 반영
  │     ├─ 컬렉션 추가
  │     └─ 연결 탭 → Sea 카메라 이동(새 노드 선택) → 시트 갱신  (탐험 지속)
  │
  ├─ 미발견(새로운 흔적) 탭 → 드러남 → 이웃 확장
  │
  └─ 스와이프 → 인접 실마리 연속 탐색
```

핵심 루프: **탐험(Sea) → 발견(Land 시트) → 저장 → 더 밝아진 세계 → 새로운 흔적 → 다시 탐험.**

## 3. 감정 목표 / 비목표

- 목표: "내 세계가 밝아진다", "저기 뭐가 있을까"(미발견의 호기심), 조용하고 정돈된 탐험.
- 비목표: 정보 정리 도구 느낌, 무한 추천 피드(SNS), 유치한 `???`/잠금 표현, 무거운 그래프 조작.

## 4. MVP 범위 / 추후

- **MVP:** Sea(노드·연결·미발견, 가벼운 레이아웃) + Land 시트(요약·저장·연결·컬렉션) + Sky(오늘의 발견·추천·미발견 점프).
- **추후:** Culture Map 모드, 탐험률 시각화 고도화, 관점 레이어, (장기) Globe.

> 구현 메모: Land 시트는 RN bottom-sheet, Sea는 RN SVG/Reanimated 가벼운 노드 그래프. 실데이터는 EXP4 이후.
