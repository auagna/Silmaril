v2 · 5탭 + Atlas 중심

  1. Home — Discovery(오늘의 발견·이어보기·큐레이터픽·신규). AI-seed 칩.
  2. Search — 타입별 실마리 찾기(독립 탭).
  3. Explore = Atlas ★ — 어두운 세계 + 저장 노드 + Fog(???) + 진행률 + 모드(Graph/Cultural Map/Globe🔒).
  4. Detail — 연결 먼저 + AI Wiki(접힘) + 관점(Should).
  5. Records — Saved/Notes/Collections/Reviews (컬렉션 여기).
  6. Profile — 탐험 정체성 + 통계 + 밝힌 세계.

눌러볼 것: Home 카드→Detail · Detail 연결→다른 실마리 · Atlas의 ???(Fog) 탭→밝히기 · Detail "Atlas에서 보기" · ＋저장.

v1 대비 변경
· Map(그래프) → Explore=Atlas 승격, 그래프는 한 모드
· Fog/진행률/문화맵 신규 (어두운 세계를 밝히는 감정)
· Create 탭 제거 · Search 독립 · Collections는 Records 안
· AI-seed 칩 표시 (AI가 먼저, 유저가 보정)
· Perspectives → Should-have(가볍게)

v2 의도: "정보 정리"가 아니라 "내 세계를 밝힌다"는 감정. OK면 D1(shadcn 토큰)으로.
