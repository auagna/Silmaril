5개 화면 · 클릭 흐름

  1. 탐험 피드 — 오늘의 앵커 + 이어보기 + 스캔 카드(저장 한 탭). P3·P4·P8
  2. 실마리 상세 — 연결 먼저, 본문 접힘, 관점 레이어. P1·P2·P6
  3. 컬렉션 — 취향의 단위, 팔로우, "짓는" 추천. P3·P5
  4. 로컬 그래프 — 실마리에서 진입, 1–2홉, 보조. P4
  5. 큐레이터 — 기여 가시성 배지, 팔로우 컬렉션. P6

눌러볼 것:

  1. 피드 카드 → 상세
  2. 상세의 연결 행 → 다른 실마리 (탐험 지속)
  3. ＋저장 → 저장 시트(정리는 선택)
  4. 상세 → 지도에서 보기 → 로컬 그래프
  5. 관점의 작성자 → 큐레이터

v1 의도: "예쁨"이 아니라 흐름 검증. 이 구조가 5개 UX 문제를 푸는지 본다. OK면 D1(디자인 토큰) → Figma → 코드.
